PlutusOSS

It is hard to get financial information (revenue, net-income, etc.) for German stocks in a format for computers - especially if you canâ€™t afford to pay some expensive financial service.

PlutusOSS is the solution: a open-source and free financial API

How to use it:

1. Create a account at plutusoss.pythonanywhere.com/signup/ and save your token

2. Discover the (browsable) API at plutusoss.pythonanywhere.com/api/

3. Start making requests to get your data. Don't forget: to authorize your request you need a custom header - key: Authorization value: "Token " + [youractualtoken]
Example: A GET request to plutusoss.pythonanywhere.com/api/stock/1/ with the header Authorization:Token 193847fky9s02kf7cm4ly92jfm4hx81jdnfmy8173.

What if I lose my token?

You can make an POST request to https://plutusoss.pythonanywhere.com/api-token-auth/ with JSON a body with your username and password
Example: 

{
    "username": "youractualusername",
    "password": "yourpassword"
}
It will return your token.
